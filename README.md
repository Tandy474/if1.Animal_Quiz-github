# if1.Animal_Quiz-github
A Python quiz game about animals.
The repository demonstrates the use of functions: Modular code, and the use of  "id", "Elif", and "Else".

Requiremts
Python 3.
VS Code - for editing and running the code

How to run in Windows VS Code
Open VS Code
  Create a new folder called My_Animal_Quiz
  Create a new file named if1,Animal_Quiz.py
  Save the file
  Open the integrated terminal in VS Code
Run the programme
https://github.com/Tandy474/if1.Animal_Quiz-github.git

How to play
  The quiz contains 5 animal questions
  Each question has 4 multiple-choice options.
  Type the number of your answer (1-4) and press enter
  The score will be displayed at the end.

Technical documentation
Functions:
  ask_question() displays a question
  run_quiz() runs the quiz loop
  Selection: Uses if, else, elif  for correct or incorrect answer
